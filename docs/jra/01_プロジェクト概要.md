# プロジェクト概要

`jra-srb` は、JRA 公式サイトの `JRADB/access*.html + cname` 導線を辿り、開催一覧、出馬表、オッズ、結果・払戻をローカル API として取得するための基盤である。

現時点では、fixture ベースのテストと JRA 実導線を併用できる構成になっている。短期目標は FastAPI として安定利用できること、中期目標は MCP から自然言語指示で確実に呼び出せることにある。

## 現在の主な機能

- 開催地単位の meeting 一覧取得
- 開催日 + 開催地 + レース番号からの出馬表取得
- 開催日 + 開催地 + レース番号からのオッズ取得
- 開催日 + 開催地 + レース番号からの結果・払戻取得
- 日本語入力の正規化
- 保存済み結果の JSONL / SQLite 検索・取得
- 開催日、開催場、キーワードからのレース検索
- 過去結果バッチの JSONL / SQLite 保存
- API からの結果収集 Job 作成・一覧・詳細確認
- upstream への同時実行数制限と最小アクセス間隔設定
- MCP HTTP 入口 (`/mcp`)
